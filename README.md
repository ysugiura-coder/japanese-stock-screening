# 国内株スクリーニングアプリ

国内株式市場の銘柄を複数の財務指標に基づいてスクリーニングするWebアプリケーションです。

## 機能

- **スクリーニング機能**: PER、PBR、ROE、配当利回り、時価総額、出来高、株価による条件検索
- **データ更新頻度の調整**: 1時間、6時間、12時間、24時間、手動更新から選択可能
- **表示件数選択**: 10、20、50、100件から選択可能
- **ソート機能**: 各指標で昇順/降順ソート
- **ページネーション**: 大量データの効率的な表示
- **CSVエクスポート**: 検索結果をCSV形式でダウンロード
- **レスポンシブデザイン**: モバイルデバイスにも対応
- **実際の株価データ取得**: Yahoo Finance APIからリアルタイムデータを取得

## 技術スタック

- **フレームワーク**: Next.js 14 (App Router)
- **言語**: TypeScript
- **スタイリング**: Tailwind CSS
- **データフェッチング**: TanStack Query (React Query)
- **株価データ**: Yahoo Finance API (yahoo-finance2)
- **デプロイ**: Vercel（推奨）

## セットアップ

### 1. 依存関係のインストール

```bash
npm install
```

### 2. 開発サーバーの起動

```bash
npm run dev
```

ブラウザで [http://localhost:3000](http://localhost:3000) を開いてください。

### 3. 別端末から見る・サーバーに載せる

- **同一ネットワーク（パブリック）**: 同じ Wi‑Fi 内のスマホ・別 PC から見る → この PC の IP で `http://<IP>:3000` にアクセス（要: ファイアウォールでポート 3000 許可）
- **どこからでも見る**: Vercel などにデプロイして URL で公開
- **自前サーバー**: VPS 等で `npm run build` → `npm run start` で常時起動

詳細は **[PUBLIC_AND_SERVER.md](./PUBLIC_AND_SERVER.md)** を参照してください。

## データソース

### Yahoo Finance API（現在使用中）

- **無料**: 完全無料で利用可能
- **データ更新**: リアルタイム
- **取得可能銘柄**: 主要な東証プライム銘柄（約50銘柄）
- **取得データ**: 株価、時価総額、出来高、PER、PBR、ROE、配当利回り

### J-Quants API（オプション）

- **無料プラン**: 1日1回更新、1,000銘柄まで
- **APIキー取得**: https://jpx-jquants.com/
- **設定方法**: `.env.local`ファイルに`JQUANTS_API_KEY`を設定

### モックデータ（フォールバック）

APIが利用できない場合、開発用のモックデータが使用されます。

## ランニングコスト

- **Vercel無料プラン**: 
  - サーバーレス関数: 100GB時間/月
  - 帯域幅: 100GB/月
- **Yahoo Finance API**: 完全無料
- **J-Quants API**: 無料プランあり

## 注意事項

- **レート制限**: Yahoo Finance APIにはレート制限があります。大量のリクエストを送信する場合は、適切な間隔を空けてください。
- **データの正確性**: 本アプリは投資判断の参考情報を提供するものであり、投資判断の責任はユーザーにあります。
- **利用規約**: Yahoo Financeの利用規約を遵守してください。
