# 全銘柄データの取得方法

## 現状

現在のアプリには**約49銘柄のモックデータ**のみが含まれています。
日本株の全銘柄は**約3,700銘柄**（東証プライム、スタンダード、グロース含む）あります。

## 全銘柄を取得する方法

### 方法1: J-Quants API（推奨）

J-Quants APIを使用すると、無料プランで**1,000銘柄まで**取得できます。

#### セットアップ手順

1. **J-Quantsアカウントを作成**
   - https://jpx-jquants.com/ にアクセス
   - 無料アカウントを作成

2. **APIキーを取得**
   - アカウント設定からAPIキーを取得

3. **環境変数を設定**
   - `.env.local`ファイルを作成
   - 以下を追加：
     ```
     JQUANTS_API_KEY=your_api_key_here
     ```

4. **API実装を有効化**
   - `lib/api/jquants.ts`のTODO部分を実装
   - J-Quants APIの仕様に合わせてデータ取得処理を追加

#### J-Quants APIの制限（無料プラン）

- 取得可能銘柄数: 1,000銘柄まで
- データ更新頻度: 1日1回
- API呼び出し制限: 1時間あたり100回

### 方法2: 有料APIサービス

全銘柄を取得するには、有料のAPIサービスが必要です：

- **J-Quants有料プラン**: 全銘柄取得可能
- **その他の金融データAPI**: 各種サービスあり

### 方法3: スクレイピング（非推奨）

- 利用規約違反の可能性
- レート制限の問題
- メンテナンスが困難

## 現在の実装状況

- ✅ モックデータ: 約49銘柄（主要銘柄）
- ⚠️ 全銘柄取得: J-Quants APIの実装が必要
- ✅ データ構造: 全銘柄に対応可能な設計

## 次のステップ

1. J-Quants APIキーを取得
2. `.env.local`にAPIキーを設定
3. `lib/api/jquants.ts`のAPI実装を完成させる
4. テストして動作確認

## 注意事項

- 無料プランでは1,000銘柄まで
- 全銘柄（3,700銘柄）を取得するには有料プランが必要
- APIの利用規約を必ず確認してください
