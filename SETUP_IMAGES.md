# 画像ファイルのセットアップ

## 手順

1. `制作イメージ画像フォルダ` 内の画像ファイルを `public/images` フォルダにコピーしてください。

### コピーするファイル

以下のファイルを `public/images` にコピー：

- `PTS_1.JPG` → `public/images/PTS_1.JPG`
- `PTS_2.JPG` → `public/images/PTS_2.JPG`
- `決算.JPG` → `public/images/決算.JPG`
- `受注.JPG` → `public/images/受注.JPG`
- `銘柄.JPG` → `public/images/銘柄.JPG`
- `リアルタイム.JPG` → `public/images/リアルタイム.JPG`

### Windowsでのコピー方法

1. エクスプローラーで `制作イメージ画像フォルダ` を開く
2. すべての画像ファイルを選択（Ctrl+A）
3. コピー（Ctrl+C）
4. `public/images` フォルダを開く
5. 貼り付け（Ctrl+V）

### 確認

コピー後、以下のパスにファイルが存在することを確認：

```
public/
  images/
    PTS_1.JPG
    PTS_2.JPG
    決算.JPG
    受注.JPG
    銘柄.JPG
    リアルタイム.JPG
```
