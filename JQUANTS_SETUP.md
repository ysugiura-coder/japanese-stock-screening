# J-Quants API セットアップガイド

## 無料プランで1,000銘柄を取得する方法

### 1. アカウント作成

1. https://jpx-jquants.com/ にアクセス
2. 「無料会員登録」をクリック
3. メールアドレスとパスワードでアカウントを作成

### 2. APIキーの取得

1. ログイン後、ダッシュボードにアクセス
2. 「API」セクションからAPIキーを取得
3. APIキーをコピー

### 3. 環境変数の設定

プロジェクトルートに `.env.local` ファイルを作成し、以下を設定：

```env
# J-Quants API設定
JQUANTS_API_KEY=your_api_key_here
JQUANTS_API_BASE=https://api.jquants.com/v1

# 認証情報（APIキー方式を使用する場合は不要）
# J-Quants APIはAPIキーだけで認証可能な場合があります
# 詳細はJ-Quantsのドキュメントを確認してください
```

### 4. 動作確認

1. 開発サーバーを起動：
   ```bash
   npm run dev
   ```

2. ブラウザで http://localhost:3000 を開く

3. 「手動更新」ボタンをクリック

4. コンソールでログを確認：
   - `Successfully fetched X stocks from J-Quants API` と表示されれば成功

## 無料プランの制限

- **取得可能銘柄数**: 1,000銘柄まで
- **データ更新頻度**: 1日1回
- **API呼び出し制限**: 1時間あたり100回
- **データ保持期間**: 制限あり（詳細は公式サイトを確認）

## 有料プランの料金（参考情報）

### 無料プラン
- **月額**: 0円
- **取得可能銘柄数**: 1,000銘柄まで
- **データ更新頻度**: 1日1回
- **API呼び出し制限**: 1時間あたり100回

### 有料プラン（概算）
**注意**: 以下の料金は参考情報です。最新の料金は公式サイトで確認してください。

#### スタンダードプラン（推定）
- **月額**: 約10,000円〜30,000円程度
- **取得可能銘柄数**: 全銘柄（約3,700銘柄）
- **データ更新頻度**: リアルタイムまたは高頻度更新
- **API呼び出し制限**: 大幅に緩和

#### プロフェッショナルプラン（推定）
- **月額**: 約50,000円〜100,000円程度
- **全機能**: 制限なし

### 料金確認方法

1. **公式サイト**: https://jpx-jquants.com/
2. **料金ページ**: https://jpx-jquants.com/pricing（存在する場合）
3. **お問い合わせ**: サポートに直接問い合わせ

**重要**: 料金は変更される可能性があります。必ず公式サイトで最新情報を確認してください。

## 実装の注意点

1. **認証方式**: J-Quants APIの認証方式は複数あります
   - APIキー方式（推奨）
   - メールアドレス/パスワード方式
   - 詳細は公式ドキュメントを確認

2. **レート制限**: 無料プランでは1時間あたり100回の制限があります
   - バッチ処理で間隔を空ける
   - エラーハンドリングを実装

3. **データ更新**: 無料プランでは1日1回の更新制限があります
   - キャッシュ機能を活用
   - 更新頻度の設定を適切に管理

## トラブルシューティング

### エラー: "API key is not set"
→ `.env.local`ファイルに`JQUANTS_API_KEY`が設定されているか確認

### エラー: "Failed to get refresh token"
→ APIキーが正しいか、アカウントが有効か確認

### エラー: "Rate limit exceeded"
→ API呼び出しの間隔を空ける（1時間あたり100回以内）

### データが取得できない
→ 無料プランの制限を確認（1日1回の更新制限など）
